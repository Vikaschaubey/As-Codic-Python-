Swap without using third variable

a=int(input("Enter the first number : "))
b=int(input("Enter the second number : "))
print("\nBefore swaping")
print(f"a = {a} \nb = {b}")
a, b=b, a
print("\nAfter swaping")
print(f"a = {a} \nb = {b}")

OUTPUT :-

Enter the first number : 25
Enter the second number : 15

Before swaping
a = 25
b = 15

After swaping
a = 15
b = 25
