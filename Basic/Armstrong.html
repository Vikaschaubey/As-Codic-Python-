Armstrong Number

n=int(input("Enter the number : "))
t=n
s=0
while(n>0):
 r=n%10
 s+=r**3
 n=n//10

if t==s:
 print(f"{t} is a Armstrong number")
else:
 print(f"{t} is not a Armstrong number")

OUTPUT :-

Enter the number : 153
153 is a Armstrong number
